# Movie App

A Flutter Movie app that uses IMDB Api, GetX for state management, and Hive for efficient local caching.
<br></br>
![Design sans titre (1) (1)](https://github.com/user-attachments/assets/138a0053-1e76-4653-ab0e-6fc790e702aa)
<br></br>
<br></br>
The app is done with passion and a focus on clean architecture and maintainable code.
It includes key features such as :
 - Internet connection check.
 - Fetches trending & top rated movies from Imdb Api with complete details.
 - Save & cache your favorite movies to access them even offline.
